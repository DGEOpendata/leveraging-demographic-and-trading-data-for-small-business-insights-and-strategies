# Leveraging Demographic and Trading Data for Small Business Insights and Strategies

## Overview
This use case aims to integrate small business demographic data with trading summary data to generate actionable insights for entrepreneurs, policymakers, and researchers. By analyzing these datasets, stakeholders can identify growth opportunities, design targeted support programs, and understand economic trends impacting small businesses.

## Prerequisites
- Python 3.7 or higher
- Pandas library for data manipulation
- Matplotlib for data visualization

## Installation

pip install pandas matplotlib


## Data Files
- `Small_Business_Demographics_and_Performance.csv`: Contains demographic and performance data of small businesses in Abu Dhabi.
- `Trading Summary by Investor Type - Jan 2020 - Nov 2025_0.xlsx`: Contains trading summary data by investor type.

## Instructions
1. Load the datasets using Pandas.
2. Perform analysis to identify top growth industries and trading volume trends.
3. Use Matplotlib to visualize the results.
